# Aula 15 - Linguagem C - Funções - Inicialização: ```c //return_type func_name(arg1_type arg1, arg2_type arg2, ...){} void anyFunction(void) {} int main(void) {} int cube(int num) {return num * num * num;} ... /* !!PROTÓTIPOS DE FUNÇÃO servem para dizer ao compilador que existem funções que estão escritas após o main() declaração: prot_type prot_name(arg's_type's) */ #include // definindo um protótipo int cube(int); int main(void) { printf("3^3 == [%d]", cube(3)); } // função escrita abaixo da main() int cube(int num) { return num * num * num; } ``` - Saídas de erro padrão ```c /* "1>" ou ">" -> saída padrão (stdout) "2>" -> saída de erro (stderr) -> entrada padrão (stdin) -> saída auxiliar (stdaux) -> saída de impressora (stdprn) retorno de processo padrão possui 2 constantes: EXIT_SUCCESS e EXIT_FAILURE. O padrão de EXIT_FAILURE é 3 (e não 1). Mas, na prática, valores diferentes de 0 são códigos de erro. */ void error(char *msg) { fprintf(stderr, "ERRO: %s\n", msg); exit(1); } int main(void) { int num; scanf("%d", &num); if(num < 1) error("mentiroso"); else fprintf(stdout, "Sua idade e: %d", num); /* !!CUIDADO!! o printf() usa um stream chamado stdout() que é bufferizado p/ linha. Assim, devemos "descarregar" esse buffer, dessa forma: // buffer ainda carregado printf("Digite: "); scanf("%d", &x); // buffer descarregado printf("Digite: "); fflush(stdout); scanf("%d", &x); */ return 0; } /* # NO BASH $ cc funcao.c -o func $ ./func 2> log.txt $ cat log.txt # output ERRO: mentiroso */ ```
