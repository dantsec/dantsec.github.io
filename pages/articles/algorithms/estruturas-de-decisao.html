# Aula 03 - Algoritmos - Estruturas de Seleção / Decisão / Condição ### Estruturas de Seleção - São comandos que auxiliam no direcionamento da sequência de execução de um programa por meio da avalização de condições lógicas - Algoritmo condicional: Permite a escolha de um grupo de ações a ser executado quando determinadas condições são satisfeitas - Para que servem? - Permitem alterar o Fluxo de Execução do algoritmo, de forma a selecionar qual parte deve ser executada - Funcionamento: - Após executar as funções de validação e comparação, as estruturas condicionais irão executar os blocos de comando definidos de acordo com o resultado da comparação - Tipos de Estruturas Condicionais: - If / Else (Se / Então) - Switch / Case (Escolha / Caso) ### Estruturas de Seleção - IF / ELSE - Classificação: - Simples - Composta - Aninhada - Estrutura de Seleção simples: ``` if do endif ``` - Estrutura de Seleção Composta ``` if { do } else { do } ``` - Estrutura de Seleção Aninhada / Encadeada ``` if { if { do } endif } else { do } endif ``` ### Estrutura de Seleção - SWITCH / CASE - É utilizada quando é necessário testar a mesma variável com uma série de valores - Estrutura Padrão: ``` switch case1: do break case2: do break // se nenhum dos cases funcionar, default executará (uso opcional) default: break endswitch ```
